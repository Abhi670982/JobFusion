# JobFusion

## 🚀 Overview

JobFusion is an AI-powered job aggregation and career management platform that brings opportunities from multiple job sources into a single dashboard. It helps users discover relevant jobs, manage applications, track progress, and optimize their profiles using resume intelligence and skill extraction.

## ✨ Key Features

* 🔍 Unified job search from multiple job sources
* 📄 AI-powered resume parsing and skill extraction
* 🎯 Personalized job recommendations
* 📊 Real-time user dashboard and analytics
* ❤️ Save and manage jobs
* 📝 Application tracking system
* 👤 Profile management and onboarding
* 🔔 Smart notifications and activity tracking
* 🔐 Secure authentication with Clerk
* ☁️ Cloud-based data storage using MongoDB Atlas
